# Kilig



## 介绍

Kilig是一个简化UITableView数据驱动的库

## 要求
iOS8、Swift3.0及以上或则Objective-C

## 安装

Kilig 支持 [CocoaPods](http://cocoapods.org) 集成安装。 请在Podfile文件中添加：

```ruby
pod 'Kilig'
```
